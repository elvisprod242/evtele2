# **App Name**: StreamVerse

## Core Features:

- Live TV Streaming: Stream live TV content using M3U8 streams, with play/pause, full screen, and volume controls.
- Live Radio Streaming: Stream live radio content with audio controls, displaying station logo and track information.
- Program Guide: Display a chronologically ordered schedule of TV and Radio shows.
- User Authentication: Allow users to create accounts and log in to access personalized features like commenting and saving favorite shows.
- Social Interaction: Users can 'like' current broadcasts, as well as post comments.
- Content Recommendations: Suggest past episodes or content based on viewing history. Use AI to pick and re-order shows for specific users, creating custom suggestions

## Style Guidelines:

- Primary color: A deep purple (#673AB7) for a modern, sophisticated feel, inspired by broadcasting and media.
- Background color: A very light gray (#F5F5F5), almost white, providing a clean, neutral backdrop that emphasizes the content.
- Accent color: A vibrant orange (#FF5722) to highlight interactive elements like buttons and calls to action.
- Headline font: 'Poppins', a geometric sans-serif for headlines and short amounts of text. Note: currently only Google Fonts are supported.
- Body font: 'PT Sans', a humanist sans-serif that complements Poppins well, ensuring readability for longer text. Note: currently only Google Fonts are supported.
- Use clear, minimalist icons for navigation and controls.
- Employ a modular design with clear sections for live streams, program guides, and user accounts.
